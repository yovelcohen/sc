## **The Common Module**

this module contains constant variables, helpful recurring utils, and the base objects for any SCR project.

****dates****

* contains several consts and utils.
* utils contains recurring utilities functions and the retry decorator which can be used to retry requests on any API
  call.
* the custom admin contains admin filters and export table to csv.
* the base contains the base custom management command, base exceptions for views, base models and base views
